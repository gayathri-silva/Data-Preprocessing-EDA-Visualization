 # Data Preprocessing, EDA & Visualization Project

## Overview

This project focuses on Data Preprocessing, Exploratory Data Analysis (EDA), and Data Visualization using Python. The main objective of the project was to clean, analyze, and visualize datasets to identify patterns, trends, and insights that support data-driven decision-making.

The project was developed as part of an academic coursework activity to strengthen practical knowledge in data analysis and visualization techniques.

---

## Features

* Data cleaning and preprocessing
* Handling missing values and duplicates
* Feature selection and transformation
* Exploratory Data Analysis (EDA)
* Statistical data analysis
* Data visualization and insight generation
* Pattern and anomaly identification

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
